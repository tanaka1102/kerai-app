# AI社員ケライ — 休まない。辞めない。文句言わない。

中小企業向けAI社員サービス。ゲーム感覚で家来を採用し、事務・営業・経理をAIが代行。

**公開URL: https://kerai-app.vercel.app**

## サービス概要

- 6種類の専門AIが業務を代行（営業・事務・経理・SNS・サポート・Web制作）
- チャットで話しかけるだけで家来が動く
- 月額0円から利用可能
- ゲーム感覚で家来を採用・強化できる「城下町」システム

## 技術スタック

- [Next.js](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS
- Vercel (ホスティング)

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy

本番環境: https://kerai-app.vercel.app

Vercel + GitHub連携による自動デプロイ。`main`ブランチへのpushで自動反映。
